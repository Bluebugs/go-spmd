How It Works

The same Go SPMD source code compiles to two WASM binaries:
-simd=true produces vectorized WASM with v128 instructions (4 pixels per iteration).
-simd=false produces scalar WASM (1 pixel per iteration).
The browser detects SIMD support and loads the optimal binary automatically.
